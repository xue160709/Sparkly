# 八卦智慧卡·世界知识分类规则

## 一、核心分类逻辑:「能量属性」+「作用方向」

| 卦名 | 能量特征 | 作用方向 | 核心关键词 |
|------|----------|----------|-----------|
| ☰ 乾 | 刚健主动 | 向外突破 | 决策、执行、领导、目标 |
| ☷ 坤 | 柔顺承载 | 向内积累 | 基础、积累、包容、责任 |
| ☳ 震 | 爆发突破 | 向外行动 | 行动、创新、快速、突破 |
| ☴ 巽 | 柔顺渗透 | 向外沟通 | 沟通、适应、灵活、渗透 |
| ☵ 坎 | 深邃智慧 | 向内学习 | 学习、思考、风险、深度 |
| ☲ 离 | 热情光明 | 向外展现 | 情绪、热情、表达、价值 |
| ☶ 艮 | 静止边界 | 向内约束 | 自律、专注、止损、边界 |
| ☱ 兑 | 喜悦分享 | 向外交流 | 分享、社交、财富、喜悦 |

## 二、分类决策树

```
第一层:判断「内向 vs 外向」
├─ 外向(向外施加影响)
│   ├─ 第二层:判断「刚 vs 柔」
│   │   ├─ 刚(强势直接) 
│   │   │   ├─ 第三层:持续性?
│   │   │   │   ├─ 持续稳定 → ☰ 乾(战略决策)
│   │   │   │   └─ 爆发突破 → ☳ 震(立即行动)
│   │   │   
│   │   └─ 柔(温和渗透)
│   │       ├─ 第三层:目的?
│   │       │   ├─ 建立关系 → ☴ 巽(沟通协作)
│   │       │   └─ 传递价值 → ☱ 兑(分享财富)
│   
└─ 内向(向内修炼提升)
    ├─ 第二层:判断「动 vs 静」
    │   ├─ 动(流动成长)
    │   │   ├─ 第三层:重心?
    │   │   │   ├─ 智识学习 → ☵ 坎(学习思考)
    │   │   │   └─ 情感管理 → ☲ 离(内观情绪)
    │   │   
    │   └─ 静(约束控制)
    │       ├─ 第三层:作用?
    │       │   ├─ 止损边界 → ☶ 艮(自律专注)
    │       │   └─ 厚积根基 → ☷ 坤(稳健积累)
```

## 四、三境精度体系

### 4.1 三境定义

| 境界 | 定义 | 时间尺度 | 操作性 | 涉及对象 |
|------|------|----------|--------|----------|
| **地境** | 可操作的具体方法 | 当天/当周见效 | 有明确步骤 | 主要是自己 |
| **人境** | 人际关系中的智慧 | 数月/数年见效 | 需灵活应用 | 自己+他人 |
| **天境** | 哲学层面的觉悟 | 一生的修行 | 只有原则 | 自己+世界 |

### 4.2 三境分布比例

- **地境**: 70% - 日常可操作的具体工具
- **人境**: 25% - 需要时间验证的人际智慧
- **天境**: 5% - 终极哲学思考

### 4.3 三境判断标准

#### 地境判断标准
**符合以下条件即为地境**:
- ✅ 有明确的操作步骤
- ✅ 能提供具体工具/公式/表格
- ✅ 可在当天或当周实践并见效
- ✅ 主要涉及个人行为改变

**地境知识示例**:
- 目标分解法、时间管理工具
- 沟通话术、学习技巧
- 情绪调节方法、专注训练

####人境判断标准
**符合以下条件即为人境**:
- ✅ 需要在人际关系中实践
- ✅ 效果需要数月才能显现
- ✅ 涉及战略、领导力、协作
- ✅ 需要较高的情商和智慧

**人境知识示例**:
- 长线思维、战略布局
- 领导力原则、团队协作
- 冲突处理、信任建立

#### 天境判断标准
**符合以下条件即为天境**:
- ✅ 超越个人和他人,关乎使命
- ✅ 难以量化,需要终身体悟
- ✅ 涉及哲学、信仰、终极追问
- ✅ 不提供工具,只有思维框架

**天境知识示例**:
- "天行健,君子以自强不息"
- "上善若水,水善利万物而不争"
- "独乐乐,不如众乐乐"

## 六、分类实战指南

### 6.1 分类四步法

**步骤1: 识别作用方向**
- 问题: 这个知识是让我「向外施加影响」还是「向内修炼提升」?
- 向外 → 进入外向卦组(乾、震、巽、兑)
- 向内 → 进入内向卦组(坤、坎、离、艮)

**步骤2: 判断能量属性**
- 外向卦: 是「刚」(直接强势)还是「柔」(温和渗透)?
  - 刚 → 乾或震
  - 柔 → 巽或兑
- 内向卦: 是「动」(流动成长)还是「静」(约束控制)?
  - 动 → 坎或离
  - 静 → 坤或艮

**步骤3: 细化具体卦象**
- 根据知识的具体特征,对照八卦关键词
- 一个知识可能符合多个卦象,标注主卦+副卦

**步骤4: 确定三境层级**
- 对照三境判断标准
- 确定这个知识属于地境/人境/天境

### 6.2 分类检查清单

**✅ 必须回答的问题**:
1. 这个知识的核心目的是什么?
2. 使用这个知识主要涉及谁?(自己/他人/世界)
3. 这个知识多久能见效?(天/月/年/终生)
4. 这个知识有具体操作步骤吗?
5. 这个知识最像八卦中的哪一个?

**❌ 常见误区**:
- 不要只看关键词,要看本质属性
- 不要强行单一归类,允许跨卦
- 不要忽略三境层级的区分
- 不要把地境知识误判为天境

## 七、跨卦处理规则

### 7.1 一知识多卦原则

**有些知识天然跨界,应该标注「主卦+副卦」**

示例:
```json
{
  "title": "第一性原理",
  "primary_gua": "☵ 坎卦",
  "secondary_gua": ["☰ 乾卦", "☳ 震卦"],
  "reason": "主要强调深度思考(坎),也涉及战略决策(乾)和创新突破(震)"
}
```

### 7.2 跨卦知识包

**建立主题包,组合多卦知识**

示例:
```
【创业者必修包】
- ☰ 乾: 长线思维、战略规划
- ☳ 震: 行动优先、快速迭代  
- ☵ 坎: 风险思维、终身学习
- ☱ 兑: 建立人脉、IP思维
```

## 八、补充原则

### 8.1 用八卦反向筛选知识

**不是「把知识塞进八卦」,而是「用八卦发现缺失的知识」**

方法:
1. 列出八卦各自最缺乏的「现代知识」
2. 主动引入西方/现代的补充内容  
3. 形成「古今中西融合」的知识体系

示例:
- ☰ 乾卦: 缺少战略工具 → 引入OKR、SWOT
- ☵ 坎卦: 缺少学习方法 → 引入费曼技巧、刻意练习
- ☱ 兑卦: 缺少财富工具 → 引入复利计算、资产配置

### 8.2 用户视角优先

**前台**: 按场景/问题分类(职场、情感、学习、财富...)  
**后台**: 用八卦体系组织,保证知识的系统性  
**关联推荐**: 基于八卦的生克关系,推荐互补的知识

## 九、质量标准

### 9.1 好的分类应该

- ✅ 逻辑自洽,可复现
- ✅ 用户能直观理解
- ✅ 覆盖完整,无遗漏
- ✅ 允许灵活,不僵化

### 9.2 避免的问题

- ❌ 过度依赖关键词,忽略本质
- ❌ 强行单一归类,丢失多面性
- ❌ 古今对立,而非融合
- ❌ 理论完美,但用户不懂

---

## 十、持续迭代

这套规则不是一成不变的,应该:
- 📊 根据用户反馈调整
- 🔄 随着知识增加而完善
- 🌱 保持开放和生长性
- 🎯 始终以用户价值为导向

---

**最后提醒**: 八卦不是「套子」,而是「生长的土壤」。规则是为了更好地组织知识,而非限制知识。
